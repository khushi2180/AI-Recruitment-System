01 — Problem Statement

Traditional Hiring Is Broken

Recruiters face three critical, systemic problems in every hiring cycle

⏱️

Time Sink

An average recruiter spends 8+ hours manually screening 100 resumes. For 300 applications, that's a full work week — before interviewing a single person.

⚖️

Unconscious Bias

Studies show candidates from Tier-1 colleges get 3× more callbacks than equally qualified Tier-3 graduates. Resume names trigger bias before a single skill is evaluated.

📉

Inconsistent Scoring

Manual evaluation inter-rater reliability is only ~60–72%. The same resume gets different scores from different recruiters on different days.

The solution: AI-powered screening that is fast, consistent, and bias-audited by design.

A 25.2-point gap that correlates with wealth and access — NOT job performance potential. This is proxy bias: the model doesn't use college directly, but rewards certifications, projects, and internships that cost money and connections.
Recommended Corrective Actions
  • Remove institution tier as implicit scoring factor
  • Add standardised technical assessment for all candidates
  • Implement name-blind CV review at manual stages
  • Reduce certification weight for roles with unequal cert access
  • Re-audit quarterly with new hire performance data
