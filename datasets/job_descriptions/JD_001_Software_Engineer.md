## Job Summary

We are looking for a passionate and talented **Software Engineer** to join our growing engineering team. As a Software Engineer at TechNova, you will work on building and maintaining high-performance, scalable web applications that serve thousands of enterprise users daily. This role is ideal for fresh graduates or early-career professionals who are eager to work in a fast-paced startup environment, learn from senior engineers, and make a real impact from day one.

You will collaborate closely with product managers, designers, and QA engineers to deliver features end-to-end. You will participate in code reviews, contribute to architectural discussions, and be expected to take ownership of your work with a high standard of quality.

---

## Key Responsibilities

- **Feature Development:** Design, develop, test, and deploy scalable software features for TechNova's core SaaS products using Python/Django or Node.js (Express) on the backend and React.js on the frontend.
- **API Development:** Build and consume RESTful APIs and GraphQL endpoints; ensure proper authentication, error handling, and documentation using Swagger/OpenAPI.
- **Database Management:** Write optimized SQL queries, design data models, and manage schema migrations using PostgreSQL and Redis for caching.
- **Code Quality:** Participate actively in peer code reviews, maintain clean, well-documented code with unit tests achieving minimum 80% coverage using Pytest or Jest.
- **CI/CD & DevOps:** Work with GitHub Actions and Docker containers; deploy services to AWS (EC2, S3, RDS) with support from senior DevOps engineers.
- **Bug Fixing & Maintenance:** Identify and resolve production bugs and performance bottlenecks within defined SLA timelines.
- **Agile Collaboration:** Participate in daily stand-ups, sprint planning, retrospectives, and backlog grooming sessions using JIRA and Confluence.
- **Documentation:** Write technical documentation for modules you develop; maintain architecture decision records (ADRs).
- **Cross-Team Collaboration:** Work alongside product, design, data, and QA teams to ensure seamless feature delivery.
- **Learning & Growth:** Attend internal tech talks, complete assigned learning paths on Udemy/Coursera, and contribute to internal knowledge-sharing sessions.
